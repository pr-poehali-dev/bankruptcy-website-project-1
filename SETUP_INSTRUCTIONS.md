# Инструкция по настройке оптимизации изображений

## Шаг 1: Скачать изображения

Выполните в терминале:
```bash
chmod +x download-images.sh
./download-images.sh
```

Это скачает все 14 изображений в папку `public/image_save/`.

## Шаг 2: Обновить конфигурацию Vite

Замените содержимое `vite.config.ts` на содержимое файла `vite.config.optimized.ts`:

```bash
cp vite.config.optimized.ts vite.config.ts
```

Или скопируйте вручную содержимое файла `vite.config.optimized.ts` в `vite.config.ts`.

## Что добавлено:

1. **Плагин оптимизации изображений**: `@vheemstra/vite-plugin-imagemin` (уже установлен)
2. **Настройки сжатия**:
   - PNG: качество 80%
   - JPEG/JPG: качество 80%
   - WebP: качество 80%

## Результат:

После сборки проекта (`npm run build`) все изображения из `public/image_save/` будут автоматически оптимизированы, что уменьшит их размер на 30-60% без потери визуального качества.

## Проверка:

После выполнения скрипта проверьте:
```bash
ls -lh public/image_save/
```

Должно быть 14 JPG-файлов.
